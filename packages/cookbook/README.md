# AI Workflow Cookbook

A practical reference guide for developers working with AI coding agents. Built with [RSPress](https://rspress.dev/).

## What's Inside

- **Core Concepts** — Vocabulary and fundamentals (rules, steering files, MCPs, agent modes)
- **Pillar 1: Prepare Your Repo** — How to structure your codebase for AI agents
- **Pillar 2: Work With Your Agent** — Spec-driven development workflow
- **Examples** — Real-world AGENTS.md files, specs, and custom commands
- **Quick Reference** — Cheatsheets and troubleshooting guides

## Development

### Setup

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open http://localhost:3000 to view the cookbook.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
