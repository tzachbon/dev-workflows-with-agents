# Resources

Curated external resources for going deeper on AI-assisted development.

## Reports and Analysis

### [2025: The Year in LLMs](https://simonwillison.net/2025/Dec/31/the-year-in-llms/)
Simon Willison's comprehensive year-end review covering reasoning models, AI agents, local models, and major trends shaping how we work with LLMs.

### [State of AI Coding 2025](https://www.greptile.com/state-of-ai-coding-2025)
Greptile's data-driven analysis of AI coding adoption. Key findings: 98% of developers use AI tools weekly, 76% increase in lines of code per developer, 33% larger pull requests.

## Practical Experiences

### [Shipping at Inference Speed](https://steipete.me/posts/2025/shipping-at-inference-speed)
Peter Steinberger on "vibe coding" in practice. Real-world experience shipping software faster with AI assistance, including model comparisons and workflow tips.

### [Boris Cherny on Building Claude Code](https://www.youtube.com/watch?v=AmdLVWMdjOk)
Interview with the creator of Claude Code on what grew his career and building AI tools at Anthropic. Insights into how coding agents are designed and built.
