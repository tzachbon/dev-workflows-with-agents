## Using These Examples

These examples are templates. Adapt them to your:

- Tech stack
- Team conventions
- Project structure
- Domain language

Don't copy verbatim. Use them as starting points.

## Quick Start

1. Pick the example that matches your need
2. Copy the structure
3. Replace the content with your specifics
4. Iterate as you learn what works

The best AGENTS.md is one you've refined over time based on what actually helps the agent.
